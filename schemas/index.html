Message Schemas

There are three types of schemas: Entities, Requests, and Responses

Entities

Entities describe objects in the SagePoint database. These include 'things' such as a resource, a company, or a project.

Show Me »

Requests

Requests describe the format of all queries you can make to the SagePoint API. An example of a request query would be 'Show me all users named Tom, please'

Show Me »

Responses

A response describes the format of all data returned from SagePoint following a request. These can include a list of resources, confirmation of an action, and all errors you may receive.
